```js
// Example
setTimeout(function(){
    console.log('Hello');
}, 3000);

```

## [setInterval()](https://www.w3schools.com/jsref/met_win_setinterval.asp)

```js
// Example
const myTimer = setInterval(function(){
    console.log('Here!');
}, 3000);

clearInterval(myTimer);

```
